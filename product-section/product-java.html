Програма курсу

  • Модуль 1:

    • - Основи Java: типи даних, змінні, вирази, оператори, рядки та масиви
    • - Основні концепції програмування на Java, такі як управління потоком, виключення, класи та об'єкти, методи та конструктори
    • - Робота з інтегрованим середовищем розробки (IDE)
    • - Робота з системою контролю версій (VCS)

  • Модуль 2:

    • - Принципи ООП та їх реалізація в Java
    • - Наслідування, інкапсуляція, поліморфізм
    • - Побудова класів та інтерфейсів
    • - Обробка виключень та створення власних виключень
    • - Розробка програм з використанням ООП підходу

  • Модуль 3:

    • - Робота з базами даних на Java: з'єднання з базою даних, виконання запитів та обробка результатів
    • - Робота з мережевими додатками: створення клієнт-серверних додатків, робота з протоколами, включаючи TCP/IP та HTTP
    • - Використання бібліотек для роботи з мережевими додатками, таких як Apache HttpClient

  • Модуль 4:

    • - Основи веб-технологій: HTML, CSS, JavaScript
    • - Використання фреймворку Spring для розробки веб-додатків на Java
    • - Створення інтерфейсів користувача з використанням бібліотеки Bootstrap
    • - Робота з базами даних на веб-додатках: використання JPA/Hibernate для роботи з реляційними базами даних
    • -Розгортання веб-додатків на сервері та налаштування середовища для розгортання.
